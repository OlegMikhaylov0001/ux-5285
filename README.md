# UX-5285 Project

Простой проект для демонстрации.

## Запуск проекта

1.  **Установите зависимости:**
    Откройте терминал в папке проекта и выполните команду:
    ```bash
    npm install
    ```

2.  **Запустите проект в режиме разработки:**
    ```bash
    npm run dev
    ```
